# BMI Calculator 🧮

A simple and interactive **Body Mass Index (BMI) Calculator** built with HTML, CSS, and JavaScript.  
This project helps users calculate their BMI based on height and weight, and provides a health category (Underweight, Normal, Overweight, or Obese).

## 🚀 Features
- Easy-to-use interface
- Real-time BMI calculation
- Displays BMI value and health status
- Responsive design for mobile and desktop

## 📂 Tech Stack
- **HTML** – Structure
- **CSS** – Styling
- **JavaScript** – Logic & Interactivity

## 🎯 How to Use
1. Enter your height (in cm) and weight (in kg).
2. Click **Calculate BMI**.
3. Instantly view your BMI result and category.
